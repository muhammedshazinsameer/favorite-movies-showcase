## Project Features

* **Technology Stack:** Built using React.js and Vite.
* **Structure:** A well-structured, single-page application (SPA) with clear sections (Header, Introduction, Movies List, Preferences).
* **Content:** Showcases the title, genre, and a personal review ("Why I Love It") for at least four favorite movies.
* **Styling:** Uses modern, light-themed CSS for high readability and visual organization.

## Run Locally

Follow these steps to set up and run the project on your local machine:

1.  **Clone the repository:**
    ```bash
    git clone [PASTE YOUR GITHUB REPOSITORY LINK HERE]
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd favorite-movies-showcase
    ```
3.  **Install NPM packages (dependencies):**
    ```bash
    npm install
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    ```
    The application will typically be accessible at `http://localhost:5173`.
